# Flickr Universal React Example

## Demo
[https://flickrtest.herokuapp.com](https://flickrtest.herokuapp.com)

## Install and Run
1. Clone the repo
2. `npm i`
3. `npm start`
4. [http://localhost:3000](http://localhost:3000)

## Production build
1. Do install stuff
2. `npm run build`
3. `NODE_ENV=production PORT=3002 npm start`
4. [http://localhost:3002](http://localhost:3002)

## Heroku deployment
Heroku should trigger npm's `postinstall` and server should automatically build its own distribution code
